# 📖 Overview

Most platforms teach you **concepts**.

Most coding platforms test your **knowledge**.

**ProjectForge** helps you **become an engineer**.

Instead of watching endless tutorials, learners complete carefully crafted roadmaps where every lesson includes interactive explanations, executable code, automated tests, mini projects, and AI-powered guidance.

Think of it as:

> **Frontend Mentor × Codecademy × Mastering Backend × GitHub Classroom × AI Mentor**

---

Learning software development today is frustrating.

You jump between:

- 📺 YouTube

## ✅ Challenge System

Each lesson contains practical challenges.

Example:

```js
Implement a function named greet(name)

Expected Output:

greet("John")

Hello John
```

Click **Run Tests** to validate your solution.
